üìà Financial Anomaly Detection Method

Feature Engineering

  • log_return: ln(price_t / price_{t-1})
  • abs_return: Absolute value of log returns
  • rolling_std_7: 7-day rolling volatility
  • rolling_std_30: 30-day rolling volatility
  • norm_return: log_return / (rolling_std_30 + Œµ)

MAD Threshold Method

Median Absolute Deviation (MAD) provides robust outlier detection:

threshold = median(residuals) + k √ó 1.4826 √ó MAD
MAD = median(|residuals - median(residuals)|)

k parameter: Controls sensitivity (1.5-4.0, default 2.5)
Higher k = fewer anomalies (stricter detection)

Stress Mask Filtering

Only flag anomalies during high volatility periods:

stress_mask = |return_t| > percentile(|returns|, q)
final_anomaly = (residual > threshold) AND stress_mask

q parameter: Stress percentile (80-99, default 90)
Higher q = only extreme volatility triggers anomalies

üèîÔ∏è Earthquake Detection Method

Autoencoder Reconstruction

Models learn to reconstruct normal seismic patterns. Large reconstruction errors indicate anomalies.

Threshold Methods

  • ROC-Optimal (Youden's J): Maximizes TPR - FPR for best balance
  • F1-Score Optimization: Optimizes precision-recall tradeoff
  • MCC Optimization: Matthews Correlation Coefficient
  • Percentile: p90-p99 of reconstruction errors
  • Custom: User-defined threshold value
